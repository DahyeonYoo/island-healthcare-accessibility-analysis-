# Raw data placement

Raw and restricted files are intentionally excluded from git.

Place source files required by the root `scripts/` pipeline here or in the
subfolders expected by each script. Important examples:

- `전라남도 유인도정보_20241231.csv`
- `AL_D158_00_20260114/AL_D158_00_20260114.shp`
- `가현이의 마스터파일.xlsx`

The shared path helper is `scripts/project_paths.py`.
