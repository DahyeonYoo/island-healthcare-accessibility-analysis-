## 0. 결론부터 (TL;DR)

- 방법론 설계는 이미 수상권이다. **2SFCA·HMADI·RF+SHAP·LISA·정책매트릭스** 조합은 2025 수상작 패턴(진단→유형화→맞춤정책 + 자체지수 브랜딩 + 공간분석 + 설명가능AI)을 정확히 따르고 있다.
- **가장 큰 구멍은 SDC 반출 데이터다.** "대충 보고 나옴" = 차별화 핵심인 카드 의료소비·SKT 실거주가 손에 없다는 뜻. 6/29 전 **1회 집중 재방문**이 이 프로젝트의 승패를 가른다.

- 남은 시간이 **약 한 달**이다. 완벽한 분석보다 **끝까지 가는 분석**이 이긴다. 아래 로드맵은 "SDC 재방문 성공" 본선과 "공개데이터만으로도 제출 가능한" 안전판을 동시에 깐다.

## 1. 심사 기준 역산 — 점수가 어디서 나오는가

| 평가항목 | 1차 | 2차 | 우리 대응 |
|---|---|---|---|
| 주제 선정의 독창성 | 10 | 10 | 도서 의료공백 + "얼마나×왜" 2축 프레임 |
| 자료 분석의 우수성 | 30 | 30 | 2SFCA·RF+SHAP·LISA 복수기법 비교 |
| 데이터 활용성 | 10 | 10 | SDC(카드·SKT) + 공개데이터 4종+ 연계 |
| 시사점·아이디어 독창성·차별성 | 20 | 20 | HMADI 지수 + 도서 유형화 브랜딩 |
| **분석 결과의 활용성·실현성·기대효과** | 20 | **30** | **정책매트릭스 + 격차해소율 시뮬레이션** |
| 요청 자료 형식의 충실성 | 10 | - | 7쪽 규격·코드·메타 충실 |

심사 구조: **1차(전문가 서면, 2배수) → 2차(전문가 70% + 국민평가 30%) → 최종(대면, 순위)**.

핵심 함의 세 가지:
1. **2차의 최고 배점은 "활용성·실현성·기대효과 30점"**이다. 분석을 멋지게 하는 것보다, "그래서 어느 섬에 무엇을 얼마만큼 투입하면 격차가 몇 % 줄어드는가"를 보여주는 게 대상을 가른다.
2. **국민평가 30%**가 있다 → 통계 비전공자도 30초 만에 이해하는 **한 장 요약(정책매트릭스 + 지도)**이 필수. 어려운 모델은 부록으로, 직관은 본문으로.
3. **"자체 지수/유형화" 브랜딩**은 4개 수상작 전부의 공통점(출산유형 테스트지·빈집지수·UHII·상권프레임워크). 우리의 **HMADI(도서 의료접근 위험지수)**가 바로 그 브랜드다 — 이름·수식·근거를 명확히.

2025 수상작 6대 성공 패턴 → 우리 매핑:

| 성공 패턴 | 우리 프로젝트 적용 상태 |
|---|---|
| 사회이슈 × 정책공백 교차점 | ✅ 도서 의료공백(전남 60.6% 무의료섬) |
| 진단→유형화→맞춤정책 3단 구조 | ✅ HMADI(진단)→K-means(유형)→매트릭스(정책) |
| 이름 붙은 산출물 | ⚠️ HMADI 정의·수식 확정 필요 |
| SDC 방문 + 외부데이터 3종+ 연계 | ❗ SDC 반출 미완 — 최대 리스크 |
| 서비스 수준 시각화 | ⚠️ 지도 3장 + 매트릭스 + (선택)대시보드 |
| 방법론 투명성(결측·정규화·검증) | ⚠️ 전처리 규칙 문서화 필요(§6) |

## 2. 냉정한 현재 진단

### 강점
- **마스터데이터가 좋다.** 유인도정보(277섬 × 126변수)는 섬 단위 PK(섬코드, 중복 0건)·좌표·의료40변수·접근성변수를 한 테이블에 갖춘 보기 드문 자료. 분석의 척추다.
- **방법론이 단단하다.** 단일기법이 아니라 복수기법 비교(다중회귀 베이스라인 + RF/SHAP 메인, Choropleth 메인 + LISA 보조)로 설계해 "분석 우수성 30점"에 강하다.
- **2축 프레임이 영리하다.** "얼마나 취약(severity, HMADI)"과 "왜 취약(type, 군집/SHAP)"을 직교축으로 분리한 정책매트릭스는 한 장에 우선순위·처방·근거를 다 담는다. 심사위원이 가장 좋아하는 페이지가 될 수 있다.

### 리스크(우선순위 순)
1. **[치명] SDC 반출 미완.** 카드 의료소비(섬 거주자가 육지에서 쓴 의료비)와 SKT 실거주는 이 분석의 유일한 "비공개·차별화" 자산이다. 없으면 공개데이터만으로 경쟁 → 변별력 급락. → §4-D3, §5 Week1·2.
2. **[치명] 결항 데이터 부재.** "2SFCA + 결항률 보정"이 방법론 셀링포인트인데 폴더에 결항 자료가 없다. 기상청/해운조합 공개자료로 반드시 보완. 없으면 "결항 보정"을 빼고 운항편수·소요시간만으로 후퇴해야 한다. → §6.3.
3. **[높음] ferry 소요시간 과대추정 + 82건 미매칭.** 중간기항지 섬에 항로 전체시간이 붙어 시하도가 600분으로 찍히는 오류. 지금 값은 **상한선(upper bound)**으로만 유효. 구간별 보정 또는 선착장 좌표 공간조인 필요. → §6.3.
4. **[높음] HMADI ↔ SHAP 순환참조 위험.** SHAP 종속변수 y가 HMADI(가중합)면, SHAP은 너희가 넣은 가중치를 되돌려줄 뿐이다. **y는 실측 기반 접근성 지표**로, HMADI는 랭킹용으로 역할 분리해야 매트릭스 직교성이 산다. → §4-D2.
5. **[중간] 분석 단위 미확정(섬 vs 읍면동).** 흑산면 하나에 흑산도·홍도·가거도·만재도가 묶이면 차별화가 증발한다. 섬 단위로 가되 다중섬 면만 분배 처리. → §4-D1, §6.1.
6. **[중간] 단면(2024) 데이터.** 유인도정보는 1시점이라 시계열·인과가 약하다. 의료공급 CSV(2023~2026 분기)와 접근성격자(2020~2023)로 **변화 추세**를 일부라도 붙이면 "시의성·깊이"가 올라간다. → §6.4·6.2.
7. **[중간] 7쪽 압축.** 분석량 대비 지면이 적다. 본문엔 결과·정책만, 방법론 디테일은 PPT/코드 부록으로. → §8.

## 4. 지금 못 박아야 할 3가지 결정

### D1. 분석 단위 = **섬 단위(섬코드 PK)**, 다중섬 면만 분배
- 유인도정보가 이미 섬 단위(섬코드 고유)다. 이걸 기본 행으로 잡으면 정합성이 깔끔하다.
- 읍면동(emd) 단위 데이터(인구등록부·격자)는 **섬으로 내려보낸다(disaggregation)**: 단일섬=면 케이스가 대부분이라 그대로 매칭되고, 흑산면·조도면처럼 다중섬인 곳만 **SKT 격자 주거인구 비율 또는 면적 비율로 섬별 분배**.
- 동명이섬 17건(개도·금호도·백야도·송도 등)은 **반드시 섬코드로 병합**, 섬이름 단독 조인 금지.

### D2. SHAP 종속변수 ≠ HMADI (순환참조 차단)
- **HMADI = 결과①(서술형 랭킹지수)**: "얼마나 심각한가" 줄세우기. 여러 변수의 정규화 가중합. P4에서 지도·랭킹으로.
- **SHAP의 y = 실측 기반 단일 접근성 지표**: 후보 — (a) 2SFCA 접근성 점수, (b) 접근성 격자 도로거리(value), (c) 카드 의료소비 기반 "의료이용 공백지수". X에는 결항률·운항편수·고령화율·공보의수·면적 등 **구성요소를 직접 넣지 않은** 수요·공급·교통 변수.
- 둘이 다른 축이어야 매트릭스(세로=HMADI severity, 가로=군집 type)가 직교한다.
- **추가 주의**: K-means 입력변수와 HMADI 구성변수가 같으면 또 직교성이 깨진다. K-means에는 **유형을 가르는 변수(교통/인력/수요 프로파일)**를, HMADI에는 **심각도 종합변수**를 쓰되 겹침을 최소화.

### D3. SDC 재방문 = **간다(1회 집중)**, 안 되면 공개데이터 안전판
- 재방문 목표를 **딱 2개로 압축**: ① 카드 의료소비(거주지=섬 법정동 × 사용지 × 의료업종 × 건수·금액), ② SKT 실거주(집계구 × 성·연령 + 좌표).
- 회의록에 이미 추출 함수 5종(build_island_codes 등)을 짜뒀다. **방문 전 코드 완성 → 현장에선 ①경로 ②도서코드목록 ③의료업종코드 3개 빈칸만 채워 실행 → 반출**. 2시간이면 끝난다.
- **반출 리스크 대비**: 인구 극소 섬은 비식별 셀 마스킹으로 반출 거부될 수 있다 → 연도합산·연령광역화(예: 65+ 단일구간)로 미리 셀 키우기.
- **안전판**: 재방문이 무산돼도 공개데이터(유인도 master + 격자 + ferry + 결항 + 의료공급)만으로 전체 분석이 닫히도록 설계(아래 로드맵은 이 구조). SDC 데이터는 "있으면 +", 없으면 빠지는 모듈로.

## 6. 데이터별 구체 전처리 가이드

> 공통: 인코딩 `utf-8-sig`, 좌표계 **EPSG:5179**(Korea 2000 Unified, 격자) ↔ 위경도는 **EPSG:4326**. 모든 조인 PK는 **섬코드(9자리)** 기준, 코드체계가 다른 자료는 매핑표 경유.

### 6.1 유인도정보 (master, 277×126)
목적: 분석 척추 테이블. 섬 1행.
- **증감률**: `'▼12.8%'` 등 문자형 + 결측 4건 → 부호 변환(`▼`=음, `▲`=양) 후 float. `증감량`(정수)이 이미 있으니 증감률은 보조로.
- **이진 변수**(상수도·송전·태양광 등 13개, 육지연결·여객선·접안시설 등): `○/X` → `1/0`. `기타/공란`은 별도 결측 플래그.
- **의료 40변수 3단 구조**(시설수/병상수/의사수 × 기관종별): 도메인별 규칙 통일. **1차의료 가용지표** = `의원+보건지소+보건진료소+보건소` 시설 합. **의료공백 플래그** = (`의원 시설 수`==0 & `보건지소`==0 & `보건진료소`==0) → 168섬(60.6%) 재현 확인.
- **동명이섬 17건**: 섬코드로만 병합. 섬이름 join 금지.
- **표본 불균형**: 신안·완도·여수·진도가 80% → 시군구 가중치/계층 분석 고려, 극소인구 섬(1~10인 48개)은 민감도 분석으로 별도 표기.
- **파생변수**: 고령화율(SDC 인구등록부 결합 시), 1차의료접근 더미, 응급취약 더미, 인구밀도(인구/면적), 로그인구(우편향 → log1p).

```python
import pandas as pd, numpy as np
m = pd.read_csv("전라남도 유인도정보_20241231.csv", encoding="utf-8-sig")
# 증감률 문자 -> float
def pct(s):
    if pd.isna(s): return np.nan
    s=str(s); v=float(s.replace('▼','-').replace('▲','').replace('%',''))
    return v
m["증감률_num"]=m["증감률"].map(pct)
# 이진화
binmap={'○':1,'X':0,'O':1,'o':1}
for c in ['상수도','송전','태양광발전','풍력발전','디젤발전','해수담수화시설']:
    m[c+"_b"]=m[c].map(binmap).fillna(0)
for c in ['육지b연결유무','여객선 유도선 운항여부','접안시설 보유']:
    m[c+"_b"]=(m[c].astype(str).str.contains('연결|운항|보유')).astype(int)
# 의료공백
m["primary_care"]=m[['의원 시설 수','보건지소 시설 수','보건진료소 시설 수','보건소 시설 수']].sum(1)
m["의료공백"]=((m['의원 시설 수']==0)&(m['보건지소 시설 수']==0)&(m['보건진료소 시설 수']==0)).astype(int)
m["pop_log"]=np.log1p(m["2024년 총인구"])
m["pop_density"]=m["2024년 총인구"]/m["지적도 기준 면적(제곱키로미터)"].replace(0,np.nan)
print(m["의료공백"].mean()*100)  # ≈60.6 재현 확인
```

구조: 전국 격자 폴리곤(약 41만 셀/연), 필드 `GID, sgg_nm_k, sgg_cd, sido_nm_k, sido_cd, value, stats_year`. `value` = **가장 가까운 시설까지 도로 이동거리**. 좌표계 EPSG:5179.

## 7. 분석 방법론 확정 (정합성 점검)

**① 실질 접근성 (2SFCA + 결항 보정)** — 차별화 1순위
- 표준 2SFCA: 1단계 각 시설 공급비 `R_j = S_j / Σ_k(P_k · W(d_kj))`, 2단계 각 섬 접근성 `A_i = Σ_j(R_j · W(d_ij))`. `S_j`=의사수/병상수(용량), `P_k`=섬 인구, `W`=거리감쇠(가우시안 등).
- **거리 d 정의(섬 특화)**: `d = 육상시간 + 여객선소요 + 결항페널티`. 결항페널티 예: `소요 × 1/(1−결항률)` 또는 기대대기시간 가산.
- **직선거리 vs 실이동시간 비교 그림**(P3)이 "실질 접근성"을 시각적으로 증명 — 점수 직결.

**② HMADI (도서 의료접근 위험지수)** — 결과①, 랭킹용
- 구성요소(예): 1차의료 접근(격자 도로거리), 응급 접근(2SFCA 응급점수), 교통(결항률·운항편수·소요시간), 수요(고령화율·인구). 각 변수 **방향 통일 후 min-max 정규화 → 가중합**.
- 가중치 근거 제시(전문가 가중 or PCA 1주성분 or 균등). HMADI와 검증지표(의료공백 더미·실의료소비)의 상관으로 타당성 입증(빈집팀이 0.852 상관으로 한 것처럼).
- **주의**: HMADI는 SHAP의 y로 쓰지 말 것(§4-D2).

**③ K-means 군집 (도서 유형)** — 결과②-a, 가로축
- 입력: 교통·인력·수요 프로파일 변수(HMADI 구성과 겹침 최소화). 표준화 후 **엘보우 + 실루엣**으로 k 결정.
- 4유형 라벨링: 교통취약형 / 인력취약형 / 수요과부하형 / 복합취약형(회의록 매트릭스와 일치).

**④ 다중회귀(베이스라인) + RF+SHAP(메인)** — 결과②-b, "왜"
- y = 실측 접근성(2SFCA 또는 격자 도로거리 또는 카드 의료공백지수). X = 수요·공급·교통 변수.
- 두 모델 일치 → 신뢰도↑, 불일치 → 토론거리. **수요 vs 공급 영향력 판정**: "응급 접근성은 섬 내부 시설이 아니라 여객선 운항·선착장 거리에 좌우" 형태로 서술.
- SHAP 워터폴을 **유형별로** 그려 군집과 원인을 연결.

**⑤ LISA (공간 자기상관)** — 보조
- Moran's I + 1개 핵심변수(예: HMADI 또는 응급접근성)로 High-High 클러스터("전남 서남부 도서 집단 위험") 발견. Choropleth가 메인, LISA가 보조, KDE는 부적합(섬=이산점).

**산출물 종합**: 지도 3장(Choropleth) + LISA 1장 + 직선/실이동 비교 + SHAP + **정책매트릭스 1장** + 시뮬레이션.

## 8. 보고서 7쪽 구성 (페이지별 설계)

| 쪽 | 제목 | 핵심 내용 | 노리는 배점 |
|---|---|---|---|
| P1 | 문제제기 + 2축 프레임 | 전남 도서 60.6% 의료공백, "얼마나×왜" 직교 프레임 | 주제 독창성 |
| P2 | 데이터·변수 + 차별화 포인트 | SDC 카드·SKT + 공개 4종 연계, "실질 접근성" 정의 | 데이터 활용성 |
| P3 | 실질 접근성 분석 | 직선거리 vs 실이동시간(여객선·결항보정) 비교 | 분석 우수성 |
| P4 | HMADI + 도서 랭킹 + 지도 | 위험지수 정의·수식·타당성, Choropleth | 분석 우수성·시사점 |
| P5 | 도서 유형 분류(K-means) | 4유형 + 유형별 특성·지도 | 시사점 차별성 |
| P6 | 유형별 음의 요인(SHAP) | 수요 vs 공급 판정, 유형별 워터폴 | 분석 우수성 |
| P7 | 정책매트릭스 + 시뮬레이션 | severity×type 처방 + 격차해소율 | **활용성·실현성·기대효과 30점** |

- **부록(PPT·코드)**: 전처리 규칙, 모델 비교표, 2SFCA 수식, 검증(교차검증·결측·Data Leakage 방지). 7쪽엔 못 담는 깊이를 여기로.
- 본문은 **결과·정책 중심 산문**, 방법은 한 줄 근거 + 부록 참조.

## 9. 차별화·가점 전략 (우승 결정 요인)

1. **브랜딩된 산출물 3종 세트**: ① HMADI(이름·수식·타당성 상관) ② 도서 4유형(직관 라벨) ③ 정책매트릭스(한 장에 우선순위·처방·근거). 2025 수상작 전부가 "이름 붙은 산출물"을 가졌다.
2. **활용성 30점 직격 — 격차해소율 시뮬레이션**: "백령형(교통취약) 상위 섬에 응급헬기 권역배치 시 응급접근성 47분→X분, 위험군 Y% 해소" 같은 **정량 효과**. 추상적 "지원 필요" 금지.
3. **국민평가 30% — 30초 요약 1장**: 비전공자도 읽는 문제→지수→매트릭스 인포그래픽. 가능하면 **간단 인터랙티브 지도/대시보드**(섬 클릭 시 HMADI·유형·처방) — 빈집팀 웹앱·소상공인팀 대시보드가 받은 가점.
4. **시의성**: **먼섬특별법**(해당 25섬) 등 최신 정책과 연결 → "정책 공백" 프레임 강화.
5. **통계 함정 짚기**: 주민등록 vs SKT 실거주 괴리, 통계청·지자체 정의 불일치류의 "현실 문제 정면돌파"가 독창성 점수를 끌어올린다(빈집팀 사례).
6. **방법론 투명성**: 결측·정규화·모델비교·검증을 명시 → 분석 우수성 + 신뢰.
