## 1. 역할·목표

당신은 **국가데이터 활용대회 데이터분석 보고서** 집필자다.  
입력 초안은 개요·불릿 위주일 수 있으나, 출력은 **일반 논문 본문**처럼 문단이 완결된 산문이어야 한다.

- **구조**: 일본 지리 논문 中村(2014) 가미고토 연구 — 「問題の所在 → 先行研究・目的 → 分析 → 政策含意」
- **문체**: `03_master/보고서_서론_배경.md`와 동일 register(격식체 산문, □ 소제목 유지)
- **윤문**: `humanize-korean` 스킬 — 사실·수치 불변, AI 패턴만 제거

### 성공 기준

1. 각 □ 절 아래 **2개 이상의 완결 문단**(명사형 종결·개요체 금지)
2. 「첫째/둘째/셋째」·「첫 번째 기대효과」 연속 나열 **최소화**(한 절당 1회 이하, 대체: 접속·인과 문장)
3. 「~을 통해」「~에 있어서」「~라고 할 수 있다」 **금지**
4. 배경(1장)과 분석(2장) **논리 중복 제거** — 배경은 문제·목적, 2장은 방법·결과
5. 표·그림은 **보조**만; 핵심 해석은 반드시 문장으로

## 3. 장별 집필 지침

### 3.1 배경 — `03_master/보고서_서론_배경.md` 우선 반영

**□ 주제 선정** (3단 논증, 각 1문단)

1. **공평성 원칙 vs 섬 현실** — 응급의료체계 공평성, 섬의 구조적 제약(배·차 환승, 부담)
2. **전남의 규모** — 유인도 277개(전국 섬의 59.5%, 한국섬진흥원 2025), 거주 인구 약 15.6만 명, 의료공백 섬 168개(60.6%)
3. **문제를 키우는 요인 세 가지** (서론 확정본 따름)
   - 인구: 65세 이상 27.18%(전국 최고, 전라남도 2025), 섬은 더 높음 → 응급 수요↑ 공급↓
   - 지리: 닥터헬기 인계점 232곳 중 70곳(30.2%, 보건복지부 2025); 먼섬특별법 43개 섬 지정(선택 인용)
   - 지표 한계: 국토통계 500m 접근성 — 216섬 무자료, 통계 사각지대

**□ 분석 필요성 및 전략**

- Guagliardo(2004), Luo & Wang(2003): 거리 중심 접근성의 한계
- 中村(2014): 거리·서비스 규모·대체 수단(헬기·원격)을 함께 봐야 함 — **한 문장으로 압축**
- 본 연구 전략: 277섬 전수, Y(복합 도달시간), 병목 분해, K-means 3계층, Moran's I/LISA, MDIS 수요 검증, S1–S3 시뮬레이션
- 목적: 취약 섬 **나열**이 아니라 **우선순위·처방·효과** 제시

**주의 (서론 교훈)**

- 「10개 중 6개」 등 **검증 안 된 비유** 금지 → 277/479(57.9%) 또는 59.5%(유인섬 기준)만
- 「상황 정보가 반영되지 않는다」 같은 **비문** 금지 → 「반영하지 못한다」
- 배경에 결과 수치(62.3분 등) **최소화** — 목적 문단에 1회만 허용

### 3.2 데이터 분석

**□ 데이터 선정**

- 분석 단위가 **섬코드 277개**인 이유(읍면동 평균의 왜곡, 동명이섬 17건)
- 6종 자료 표 1개 + 각 자료 **1문장** 역할 설명(불릿 나열 금지)
- MDIS 미충족의료율 정의 + 대회 SDC/MDIS 필수 요건 충족 명시

**□ 데이터 분석**

- 네 단계를 **시간 순 산문**으로: (1) 공간조인·crosswalk (2) Y 정의·가정 (3) RF/SHAP·병목·K-means (4) 지도·시나리오·Moran·MDIS
- Y 공식 코드블록 1개 유지
- 여객선 실측 r≈0.2, 시하도 600분 사례 — **검증 한계**로 짧게
- 방법 약어(STEP3–6)는 괄호 1회만; 이후 한글 풀어쓰기

**□ 분석 결과 및 해석**

반드시 다룰 결과 축(문단 순서 권장):

1. **심각도**: 평균·중앙값·60분 초과·상위 5섬 표 또는 문장 열거
2. **병목 이중 구조**: 육지지배 다수 vs 해상지배 극취약 — 20.9/41.6분
3. **SHAP**: 먼섬법 변수 제외 이유; 고립 대리변수 해석
4. **군집**: C0/C1/C2 — 표 1개 + 각 군집 **해석 문단**
5. **공간 집적**: Moran's I, HH 31, 권역 단위 투자 시사
6. **수요 검증**: MDIS 3개군 vs 전남 평균
7. **정책 시뮬레이션**: S1(초과 섬 수 불변) vs S2/S3 — 표 + S3 51.6%/65.8%/26.3%

그림 1–8: 기존 `island-medical-access/outputs/figures/` 경로 유지, 캡션 완결문.

### 3.3 분석 활용 전략

**□ 기대효과**

- 「첫 번째/두 번째」 대신 **주제문 단락**: (1) 사각지대 해소 216→0 (2) 병목 유형별 처방 분화 (3) S3 정량 효과
- 中村(2014) 인용은 **정책 차별화** 맥락에서 1회

**□ 방향제시**

- C2 → C1 → C0 → 해역권(LISA) → 한계·후속 연구 순 **산문**
- 한계: 단면·직선거리·시군구 MDIS·정책 파라미터 설계값
- 마지막 문장: 기여(전수 복원 + 진단–원인–처방–효과 연결) — **「~을 통해」 없이**

## 4. humanize-korean 적용 규칙 (요약)

| 제거·완화 | 대체 |
|-----------|------|
| 번역투 (~에 대해, ~하는 것이 중요하다) | 능동·구체 동사 |
| 기계적 병렬(첫째…넷째…) | 문단 전환·인과(그러나, 이와 달리, 한편) |
| 접속사 문두 남발(따라서, 또한) | 절 반 이하 |
| 균일한 「~하였다」 종결 | 「~이다/~된다/~보인다」 혼합 |
| 피동 남용 | 능동 우선 |

**의미 불변**: 수치·고유명사·법령명·인용문·수식·그림 경로.
